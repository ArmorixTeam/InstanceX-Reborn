# Issues

This tweak is still in beta and may be unstable or cause springboard crashes.

If you encounter any issues, please make an issue in this github repo. Attach any crash logs if the springboard crashs.

# Limitations

- Games that require intense memory usage, such as Roblox, may crash on lower-end devices.
- Instance configurations aren't saved.

# Building from source

> Github Actions

  1. Fork the repo.
  2. Navigate to Actions at the top.
  3. Allow workflows.
  4. At the side there should be "Build Theos Tweak". Tap on it.
  5. Tap "Run Workflow".
